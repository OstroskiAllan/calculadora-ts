# Calculadora TypeScript

Este é um projeto de calculadora simples feito em TypeScript no curdo da udemy da HCode. Ele permite realizar operações básicas de adição, subtração, multiplicação e divisão.

## Funcionalidades

- **Adição:** Soma dois números.
- **Subtração:** Subtrai o segundo número do primeiro.
- **Multiplicação:** Multiplica dois números.
- **Divisão:** Divide o primeiro número pelo segundo (não permite divisão por zero).

## Como Configurar e Iniciar o Projeto

1. **Pré-requisitos:**
   - Certifique-se de ter o Node.js instalado em sua máquina.
   - Clone este repositório para o seu ambiente local.

2. **Instalação:**
   - Abra o terminal na pasta do projeto.
   - Execute o comando `npm install` para instalar as dependências.

3. **Execução:**
   - Execute o comando `npm run dev` para iniciar o servidor local.
   - Acesse a calculadora no navegador em `http://localhost:3000`.
